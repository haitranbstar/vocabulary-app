# 📘 TÀI LIỆU CHỨC NĂNG HỌC NGỮ PHÁP & BÀI TẬP

------------------------------------------------------------------------

# 1️⃣ CHỨC NĂNG HỌC NGỮ PHÁP

## 1.1. Thêm chức năng học các thì ngữ pháp

### 📌 Mô tả chức năng

-   Thêm module **Học Ngữ Pháp**
-   Dữ liệu các thì được đọc từ file `grammar.txt`
-   Mỗi thì bao gồm:
    -   Tên thì (Tiếng Anh)
    -   Công thức
    -   Cách dùng
    -   Dấu hiệu nhận biết
    -   Ví dụ minh họa

------------------------------------------------------------------------

## 1.2. Cấu trúc dữ liệu trong `grammar.txt`

Ví dụ:

Tense Name: Present Simple\
Structure: S + V(s/es)\
Usage: - Diễn tả thói quen\
- Sự thật hiển nhiên\
Signal Words: - always, usually, often\
Examples: - I go to school every day.\
- She likes coffee.

------------------------------------------------------------------------

## 1.3. Hiển thị chi tiết mỗi thì

Khi click vào một thì:

Hiển thị:

-   📌 Tên thì\
-   🧩 Công thức\
-   📝 Cách dùng\
-   🔎 Dấu hiệu nhận biết\
-   💬 Ví dụ\
-   🎯 Ứng dụng trong giao tiếp hàng ngày\
-   🤖 Tạo hội thoại AI

------------------------------------------------------------------------

## 1.4. Ứng dụng các thì trong giao tiếp hàng ngày

### 🗣 Present Simple trong giao tiếp:

-   Giới thiệu bản thân\
-   Nói về thói quen\
-   Hỏi về công việc

Ví dụ: - What do you do?\
- I work in marketing.\
- I usually wake up at 6 AM.

------------------------------------------------------------------------

## 1.5. Tích hợp AI Gemini tạo hội thoại

### 📌 Prompt gửi Gemini

Create a short daily conversation (6-10 sentences)\
using Present Simple tense.\
Topic: Daily routine.\
Use natural spoken English.\
Highlight the verbs in Present Simple.

------------------------------------------------------------------------

# 2️⃣ CHỨC NĂNG BÀI TẬP NGỮ PHÁP

## 2.1. Tạo bài tập trắc nghiệm theo từng thì

### 📌 Prompt tạo bài tập

Create 20 multiple choice questions\
for Present Simple tense.

Requirements: - 4 options (A, B, C, D)\
- Only one correct answer\
- Provide answer key\
- Provide short explanation\
- Level: Beginner to Intermediate

------------------------------------------------------------------------

## 2.2. Bài tập tổng hợp tất cả các thì

### 📌 Prompt tạo bài tập tổng hợp

Create 20 mixed multiple choice questions\
covering these tenses: - Present Simple\
- Present Continuous\
- Past Simple\
- Present Perfect\
- Future (will / going to)

Requirements: - 4 options (A, B, C, D)\
- Provide answer key\
- Provide explanation\
- Random order

------------------------------------------------------------------------
